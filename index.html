It is a long established fact that a reader will be distracted by the readable content of a page when looking at its layout.

The basic difference between CSS Grid and CSS Flexbox is dimension. CSS grid is used to work with more than one dimension whereas the CSS Flexbox is used to work with a single dimension like either left-right or top-bottom

Bootstrap is a component-based responsive CSS framework. Where we will have some limitations on modifying the components. On the other hand "Tailwind CSS" is more flexible in modifying the elements. It has huge utilities to modify elements.

A box consisting of margin, padding, and border which wraps around all elements in HTML is called a box model. It actually talks about the layout and the design of HTML.

The semantic refers to the meaningful structure of HTML. Instead of using "div" if you use "header" or "nav" or "main", that makes a meanig that this section belongs to navigation or main content or to the header section. These semantic structure is more helpful to developer and search engine.
